# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project adheres to Semantic
Versioning.

## [0.1.1] - 2026-04-01

### Added

- Added generic OAuth2 helpers, token-source adapters, and runtime `oauth2:`
  support.
- Added interactive OAuth browser and exact loopback callback helpers.

### Changed

- Clarified that loopback callback capture is optional and falls back to manual
  paste-back when the callback-listener dependencies are absent.
